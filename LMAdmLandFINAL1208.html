Overview

 

Content

An introduction to MyLearning Administration

 

Content Objects and Packages: How to add or edit training item objects and packages. Includes AICC and SCORM settings

Manage User Learning

  

Questions and Exams: Create exams with objectives and questions.

Modify a Learner's Training History: Add/remove/modify course credit. Includes removing credit given in error.

 

Surveys: Create a survey and survey questions to gather learner evaluations.

Add or Remove Training: Add or Remove training from a learner's lesson plan.

  

Tasks and Observation Checklists: Create tasks to track a learner's demonstration or observation outside of MyLearning.  Checklists are two or more tasks.

Assignment Profiles: Create rules to assign training to specified learners by their ID or MyLearning profile data.

  

 People

Class Registration: Register and unregister learners for a class.

   Instructors and Proctors: See how to create instructors and proctors and assign them to learning items.

Learning Activities

   Cohorts: Create a group of related learners for running reports, scheduling, and adding curricula as a group.  In Plateau, these were known as "Classes"
Create Training Items: Create new CBTs, instructor led classes and blended courses.  Includes creating new versions of existing training and adding substitutes to learning items.  

System Administration

Classes: Create and edit classes and reserve seats

   Direct Links: Generate a URL that links directly to a learning item.

Curricula: Create a curriculum to set the retraining and required by dates of learning items

  

Administrators: Create MyLearning Administrators and assign roles to admins.

Curriculum Requirements: Used in cases where learners need to complete a certain number of hours of training to satisfy a training curriculum.

  

Reports

Programs: A syllabus of courses, a set of learning experiences, or combined learning events to create a learning path.

   MyLearning Reports: Overview of custom and standard reports.

External Links and Collections: Supplement learning material with links to external web sites. A group of External Links is a Collection.
